# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 15.x    | ✅ Supported       |
| < 15.0  | ❌ Not supported   |

## Reporting a Vulnerability

This script runs **entirely client-side** in your browser. It does not:

- Send data to any external server
- Collect personal information
- Modify system files
- Communicate with any third-party API

### Data Storage

The script uses **localStorage** (browser storage) to cache learned answers. This data:
- Stays on your device
- Is never transmitted over the network
- Can be cleared at any time via the script's "🗑 CLEAR" button

### Responsible Disclosure

If you discover a security issue related to this script, please:

1. **Do not** create a public issue immediately.
2. Email or DM the maintainer via GitHub.
3. Allow 48 hours for a response before any public disclosure.

### Scope

This script is an **educational/professional tool** designed for use on exams the user has legitimate access to. Any misuse (cheating, unauthorized access, etc.) is the sole responsibility of the end user.
